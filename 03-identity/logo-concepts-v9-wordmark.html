TYPE LOUNGE WORDMARK v9

심볼(솔리드 팬)을 걷어내고 영문 워드마크 단독으로 재설계. "분위기는 하나, 용도는 여럿"의 겹침(Overlapping) 컨셉을 글자 안에서 표현합니다 — 두 단어/면/글자가 겹쳐 만나는 지점이 곧 라운지(공유의 핵). 순수 잉크 모노크롬, 가로·세로 락업 병행.

참고 레퍼런스: NOMAD(겹친 레터), HIDE·ARCHITECH(기하·네거티브), EATON(기하 구성), negative SPACE / 폰트 = Jost(기하 산세, Futura 계열) · 일부 Archivo Black

1 · 단어 겹침 (Multiply)

TYPE와 LOUNGE 두 단어가 실제로 겹쳐, 겹친 면이 짙어짐 — 가장 직접적인 '겹침'
A1 — 세로 적층 · 겹침
TYPE 위 LOUNGE, 두 줄이 세로로 맞물려 겹침
A2 — 가로 · 이음새 겹침
한 줄, TYPE 끝과 LOUNGE 시작이 겹쳐 만남
A1 — 다크 반전
잉크 배경 위 크림/화이트

2 · 겹친 O (Venn)

두 개의 원이 겹쳐 교집합(라운지)을 만든다 — 가장 단순하고 보편적인 '겹침' 기호
B1 — 가로 · 겹친 O 이음
두 단어 사이를 겹친 두 원이 잇는다
B2 — 세로 · O 정렬 Venn
두 단어의 O를 세로로 정렬해 겹친 원으로
B1 — 솔리드 변형(엑센트)
교집합만 오렌지로 — 색은 최소 1점

3 · 면·기하 겹침 (Plane / Geometric)

기하 면이 워드마크를 가로지르거나, 글자가 면으로 구성 — 옛 '팬' 모티프의 잔향
C1 — 기울어진 면 스윕
반투명 면이 워드마크를 비스듬히 겹쳐 지난다
C2 — 헤비 기하 + 겹친 면 구분자
Archivo Black, 두 단어 사이 겹친 두 사각면
C3 — 세로 헤비 + 겹친 면
세로 적층, 줄 사이를 겹친 면이 연결

4 · 미니멀

가장 절제된 한 방 — 겹침은 단 하나의 작은 신호로
D1 — 트래킹 와이드 + 겹침 점
넓은 자간, 가운뎃점이 겹친 두 원
D2 — 세로 미니멀
TYPE / LOUNGE, 사이에 겹친 점
D1 — 다크
잉크 배경

5 · 타입 보이스 대비 (Serif × Sans)

겹침 대신 — 한 워드마크 안에 두 목소리. "하나의 시스템, 여러 용도"를 타입의 다양성으로
E1 — TYPE(산세) · LOUNGE(세리프)
기하 산세 + 웜 세리프, 가로 한 줄
E2 — TYPE(세리프) · LOUNGE(산세)
대비 반전 — 세리프가 앞
E3 — 세로 적층 대비
산세 위 / 세리프 아래, 가운데 정렬
E4 — 대문자 세리프 + 산세 (한 톤)
둘 다 대문자, 세리프/산세 굵기만 대비
E5 — 대비 + 겹침 (두 컨셉 결합)
산세 TYPE · 세리프 Lounge가 살짝 겹침
E1 — 다크
잉크 배경
읽는 법 — 두 갈래(겹침 / 타입 대비) × 가로·세로
  1. 1 단어 겹침(멀티플라이) — 두 단어를 실제로 포개 겹친 면이 짙어짐. 컨셉을 가장 문자 그대로 표현. (A1 세로 / A2 가로)
  2. 2 겹친 O(Venn) — 두 원의 교집합 = 공유 라운지. 단순·보편. (B1 가로 이음 / B2 세로 O정렬 / B 오렌지 1점 변형)
  3. 3 면·기하 겹침 — 폐기한 '팬'의 면 모티프를 워드마크에 잔향으로. 헤비 기하(Archivo). (C1 스윕 / C2·C3 겹친 면 구분자)
  4. 4 미니멀 — 겹침을 작은 점 하나로 압축, 가장 절제. (D1 가로 / D2 세로)
  5. 5 타입 보이스 대비(Serif × Sans) — 겹침 대신, 한 워드마크 안 두 목소리로 '용도의 다양성'. 산세=기능·구조 / 세리프=온기·라운지. (E1·E2 가로 / E3 세로 / E4 대문자 / E5 대비+겹침 결합)
