# Phase 3 — 컬러 팔레트 (v3, MCM 그래픽)

> 2026-06-20 개정. **미드센추리 그래픽**(바우하우스·스위스 원색) 기반의 볼드·웜 팔레트.
> 그루비/사이키델릭은 배제 — 가구(바실리·체스카·바르셀로나=바우하우스)와 같은 DNA로 정렬.
> 참고: furn.·RETROTIKA(정제된 볼드)를 앵커, THE SPIN ROOM(그루비)은 제외.
> **HEX/RGB는 디지털 확정값**, **CMYK/Pantone은 근사** — 인쇄·사이니지 발주 전 실물 스와치 확인 필수.

## 1. 팔레트 구조

- **스파인(주축)**: 번트 오렌지 · 크림 · 잉크 블랙 — 브랜드를 즉시 식별시키는 3색.
- **세컨더리 블록**: 포레스트 그린 · 코발트 블루 — 바우하우스 원색 계열 변주.
- **물성 액센트**: 브라스(놋쇠) · 웜우드 — 사진·공간 사이니지 물성.
- **포인트**: 레드 — 에디토리얼 강조(소량).

> **레이어 분리 원칙**: 컬러블록(오렌지·그린·코발트)은 **디지털·마케팅**에서 과감하게.
> **실제 공간 사이니지**는 우드·브라스·블랙 물성 중심으로 차분하게(가구를 이기지 않도록).

## 3. 사용 비율 가이드

```
스파인 (orange + cream + ink)   ████████████████  ~65%
세컨더리 블록 (green / cobalt)   █████             ~20%
물성 (brass / walnut, 사진)      ███               ~10%
포인트 (red)                     █                 ~5%
```

- 한 화면에 컬러블록은 **1~2개**까지. 색을 다 쓰지 말 것(쨍하되 난잡하지 않게).

## 4. 컬러블록 조합 (검증된 페어)

| 배경 | 텍스트/요소 | 용도 |
|---|---|---|
| orange | cream / ink | 메인 키비주얼 |
| ink | cream / orange | 야간·포스터 |
| cream | ink / orange | 본문·정보 |
| green | cream | 변주 블록 |
| cobalt | cream | 변주 블록 |

## 5. 접근성 (대비)

| 조합 | 용도 | 대비 |
|---|---|---|
| ink on cream | 본문 | ✅ AAA |
| cream on ink | 반전 | ✅ AAA |
| cream on orange | 큰 텍스트·로고 | ✅ 양호(AA, 대형만) |
| ink on orange | 라벨 | ✅ 양호 |
| cream on green / cobalt | 라벨·제목 | ✅ AA |

> orange·brass·red는 **작은 본문 텍스트 색 금지**(대비 부족). 면·블록·대형 타이포·로고에만.
