# Phase 3 — 컬러 팔레트 (v3, MCM 그래픽)

> 2026-06-21 개정. 팔레트 구조 재편 — **브랜드 식별 3색 = 메인(크림·잉크) + 엑센트(오렌지)**. 레트로 레드는 오렌지와 겹쳐 **삭제**. '물성'→'재질(물성)'로 풀어 설명.
> 2026-06-20 개정. **미드센추리 그래픽**(바우하우스·스위스 원색) 기반의 볼드·웜 팔레트.
> 그루비/사이키델릭은 배제 — 가구(바실리·체스카·바르셀로나=바우하우스)와 같은 DNA로 정렬.
> 참고: furn.·RETROTIKA(정제된 볼드)를 앵커, THE SPIN ROOM(그루비)은 제외.
> **HEX/RGB는 디지털 확정값**, **CMYK/Pantone은 근사** — 인쇄·사이니지 발주 전 실물 스와치 확인 필수.

## 1. 팔레트 구조

- **브랜드 식별 3색 = 메인 + 엑센트**:
  - **메인**: 크림 · 잉크 블랙 — 바탕과 글자를 이루는 기본 2색.
  - **엑센트**: 번트 오렌지 — 브랜드를 즉시 알아보게 하는 강조 1색.
  - → 이 3색(크림·잉크·오렌지)이 브랜드를 식별시키는 핵심.
- **세컨더리 블록**: 포레스트 그린 · 코발트 블루 — 바우하우스 원색 계열 변주.
- **재질(물성) 포인트**: 브라스(놋쇠) · 월넛(웜우드) — 사진·공간 사이니지에서 실제 소재의 질감으로 들어가는 색. ('물성'은 소재가 가진 질감·재질을 뜻하는 말.)

> **레이어 분리 원칙**: 컬러블록(오렌지·그린·코발트)은 **디지털·마케팅**에서 과감하게.
> **실제 공간 사이니지**는 우드·브라스·블랙 재질(물성) 중심으로 차분하게(가구를 이기지 않도록).

## 2. 컬러 정의

> 토큰 키 이름(`spine/*`)은 호환을 위해 유지하되, 역할은 **메인/엑센트**로 읽는다.

| 토큰 | 이름 | HEX | RGB | CMYK (근사) | Pantone (근사) | 역할 |
|---|---|---|---|---|---|---|
| `spine/orange` | 번트 오렌지 | `#E2531E` | 226, 83, 30 | 3·80·95·0 | 1655 C | **엑센트**(강조 컬러블록) |
| `spine/cream` | 크림 | `#F4E9C8` | 244, 233, 200 | 5·7·24·0 | 7499 C | **메인**(밝은 배경·레터링) |
| `spine/ink` | 잉크 블랙 | `#16130F` | 22, 19, 15 | 70·68·72·85 | Black 6 C | **메인**(텍스트·블록) |
| `block/green` | 포레스트 그린 | `#235C42` | 35, 92, 66 | 78·40·76·30 | 7727 C | 세컨더리 블록 |
| `block/cobalt` | 코발트 블루 | `#1B4FA0` | 27, 79, 160 | 92·70·6·0 | 2132 C | 세컨더리 블록 |
| `material/brass` | 브라스 | `#B0883C` | 176, 136, 60 | 28·44·90·8 | 7555 C / 메탈릭 871 C | 재질(물성) 포인트 |
| `material/walnut` | 웜우드(월넛) | `#8A5A33` | 138, 90, 51 | 35·63·85·25 | 7567 C | 재질(물성) 포인트 — 가구·우드 사인 |

## 3. 사용 비율 가이드

```
메인+엑센트 (cream + ink + orange)   █████████████████  ~70%
세컨더리 블록 (green / cobalt)        █████              ~20%
재질(물성) (brass / walnut, 사진)     ███                ~10%
```

- 한 화면에 컬러블록은 **1~2개**까지. 색을 다 쓰지 말 것(쨍하되 난잡하지 않게).

## 4. 컬러블록 조합 (검증된 페어)

| 배경 | 텍스트/요소 | 용도 |
|---|---|---|
| orange | cream / ink | 메인 키비주얼 |
| ink | cream / orange | 야간·포스터 |
| cream | ink / orange | 본문·정보 |
| green | cream | 변주 블록 |
| cobalt | cream | 변주 블록 |

## 5. 접근성 (대비)

| 조합 | 용도 | 대비 |
|---|---|---|
| ink on cream | 본문 | ✅ AAA |
| cream on ink | 반전 | ✅ AAA |
| cream on orange | 큰 텍스트·로고 | ✅ 양호(AA, 대형만) |
| ink on orange | 라벨 | ✅ 양호 |
| cream on green / cobalt | 라벨·제목 | ✅ AA |

> orange·brass는 **작은 본문 텍스트 색 금지**(대비 부족). 면·블록·대형 타이포·로고에만.

## 6. 소재 매핑 (→ Phase 4 사이니지)
