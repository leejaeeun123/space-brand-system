# Phase 3 — 컬러 팔레트

> 무드보드 게이트(충돌 0) 통과 후 확정. 확보 가구 무드에서 직접 도출.
> **HEX/RGB는 디지털 확정값**, **CMYK/Pantone은 근사값** — 인쇄·사이니지 발주 전 **실물 스와치 확인 필수**.

## 1. 팔레트 개요

웜 라이트우드 + 웜 크림을 **베이스(다수 면적)**, 잉크 블랙을 **앵커**, 코냑·크롬 실버를 **액센트(소수)**.
→ 웜/쿨 대비를 면적 비율로 통제(베이스 70% · 앵커 20% · 액센트 10% 가이드).

## 2. 컬러 정의

| 토큰 | 이름 | HEX | RGB | CMYK (근사) | Pantone (근사) | 역할 |
|---|---|---|---|---|---|---|
| `base/oat` | 웜 크림(오트) | `#F2EBDD` | 242, 235, 221 | 5 · 7 · 16 · 0 | 9224 C | 배경 다수 |
| `wood/honey-oak` | 허니 오크 | `#C49A6C` | 196, 154, 108 | 25 · 40 · 62 · 2 | 7569 C | 베이스 우드 |
| `accent/cognac` | 코냑(가죽) | `#8A553A` | 138, 85, 58 | 38 · 68 · 80 · 33 | 1685 C | 따뜻한 액센트 |
| `ink/black` | 잉크 블랙 | `#1B1A17` | 27, 26, 23 | 67 · 62 · 68 · 72 | Black 6 C | 텍스트·앵커 |
| `metal/chrome` | 크롬 실버 | `#AEB2B5` | 174, 178, 181 | 33 · 25 · 24 · 3 | Cool Gray 5 C / 877 C(메탈릭) | 쿨 액센트·메탈 |

## 3. 사용 비율 가이드

```
베이스 (oat + honey-oak)  ████████████████████  ~70%
앵커   (ink black)        ██████                ~20%
액센트 (cognac + chrome)  ███                   ~10%
```

## 4. 접근성 (대비)

| 조합 | 용도 | 대비 |
|---|---|---|
| ink black on oat | 본문 텍스트 | ✅ 충분 (≈ 13:1, WCAG AAA) |
| ink black on honey-oak | 제목·라벨 | ✅ 양호 (≈ 5:1, AA) |
| oat on ink black | 반전 텍스트 | ✅ 충분 |
| chrome on oat | ⚠️ 텍스트 부적합 | 장식·라인에만 |

> 코냑·그레이지·크롬은 **텍스트 본문 색으로 사용 금지**(대비 부족). 면·라인·라벨 배경에만.

## 5. 다크/반전 운용

- 저녁 모임(GATHER) 모드의 무드를 위해 **잉크 블랙 배경 + 오트/페이퍼 글로우 텍스트** 반전 조합 허용.
- 사이니지 야간 시인성 확보 시 반전 우선.

## 6. 소재 매핑 (사이니지·물성 연결 → Phase 4)

| 컬러 토큰 | 대응 물성 |
|---|---|
| honey-oak | 오크/자작 원목, 우드 사인 |
| ink black | 블랙 도장 스틸, 무광 아크릴 |
| metal/chrome | 헤어라인 스테인리스, 브러시드 메탈 |
| glow/paper | 한지·아크릴 확산판(백릿 사인) |
