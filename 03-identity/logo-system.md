## 2. 시그니처 — 레이어드 E

- TYPE의 **E에서 세로 스템(│)을 제거**하고 **가로 3선만** 남긴다 → “층을 쌓는(레이어드)” 개념을 워드마크에 내장.
- **3선 동일폭**(B안 확정). 위·중간·아래 3개 막대가 같은 길이 → 일반 E보다 “쌓인 층”으로 읽힘.
- 문맥(TYP_ + LOUNGE)에서 자연스럽게 **“TYPE”로 읽힌다**.
- **의미 이중성**: 3선 = 3타입(WORK·CLASS·GATHER)이 겹겹이 쌓인 하나의 공간.

### E 커스텀 사양 (vs Paperlogy 원본 E)
- 대문자 cap-height 기준 3개 가로 막대, **세로 스템 없음**.
- **막대 두께 = 10.5** (viewBox `0 0 56 70` 기준; 본체 600 스템과 무게 일치. cap×0.15).
- 막대 y 위치: `1.75 / 29.75 / 57.75` (두께 10.5 — 기존 0/28/56 슬롯 중앙 정렬).
- 폭 = 56 (viewBox), 렌더 시 `height:.70em · width:.56em`.
- **D와의 광학 여백** `margin-left: 0.115em` — 시스템 자간(−2%)과 별개로 E만 살짝 띄운다(붙어 보임 방지).
- 최종 벡터화 시 이 비율로 **커스텀 아웃라인**(패스) 제작 → 폰트 의존성 제거.

## 3. 워드마크 · 락업

- **영문** `TYPE LOUNGE` · Paperlogy **600** · 자간 **−2%**.
- **국문** `타입라운지` · Paperlogy **600** · 자간 **0%**.
- **프라이머리 = 세로 2줄** `TYPE` / `LOUNGE` · **가운데 정렬** (레이아웃 원칙과 일치).
- **베리에이션 = 가로 1줄** `TYPE LOUNGE` — 배너·웹 헤더·명함용.
- 국·영 병기 시 국문 위/영문 아래(또는 맥락에 따라 택1). 사이니지는 영문 워드마크 중심.

## 4. 컬러 사용

| 배경 | 워드마크(국·영·3선) |
|---|---|
| 화이트/라이트 | 잉크 `#16130F` |
| 잉크/다크 | 화이트 `#ffffff` |
| 오렌지 `#E2531E` | 화이트 `#ffffff` |

※ 라이트/다크 버전은 **형태·비율·정렬 100% 동일**, 채움색만 반전. 3선도 본체와 같은 색.

## 5. 클리어스페이스 · 금지

- **클리어스페이스**: 대문자 높이(cap)의 약 1/2을 사방 여백으로.
- **하지 말 것**: 비율 변형/기울임, 3선 개수·두께·간격 임의 변경, E에 세로 스템 되살리기, 채움색 외 컬러 적용, 3선을 서로 다른 폭으로(동일폭 원칙 위반), 폰트 교체.

## 6. 사이니지 적용

- 아크릴 사이니지 표준: 상단 `3F · 멀티유즈 라운지` / 중앙 `TYPE LOUNGE`(세로 2줄) / 하단 `WORK · CLASS · GATHER`.
- 반투명 아크릴 + 트레이싱지 인쇄 부착/투명 스티커(각인 비용 대체) — “레이어드” 질감과 개념적으로 일치.
- 자세한 소재·레이어드 방식 → `05-design-system/concept-keywords.md` 참조.

## 8. 벡터 에셋 (납품용 · 2026-07-05 아웃라인 완료)

> Paperlogy 600 글리프를 **패스로 아웃라인 변환**(fontTools) + 레이어드 E(cap 700 기준 3선 rect) 결합.
> **폰트 의존성 없음**(순수 path/rect) → 인쇄·간판·웹 그대로 사용 가능.

| 파일 | 용도 |
|---|---|
| `logo-wordmark-h-ink.svg` · `-white.svg` | 가로 1줄 (잉크 / 화이트) |
| `logo-wordmark-v-ink.svg` · `-white.svg` | 세로 2줄·가운데 정렬 (잉크 / 화이트) |
| `logo-wordmark-h.svg` · `logo-wordmark-v.svg` | currentColor (웹·CSS `color`로 제어) |

- 좌표계: em **900**, cap **700**. E 3선 = 두께 **105**(cap×0.15) · y **0/297.5/595**(하/중/상, cap기준) · 폭 **504** · E 앞 광학여백 **45**(P의 열린 오른쪽에 맞춰 축소; 구 MODE=103.5) · 자간 **−18**(−2%) · **T-Y 커닝 +65**(겹쳐 붙어보임 보정) · E→다음단어 갭 **666**.
- 재현: `gen_wordmark.py`(입력 `Paperlogy-6SemiBold.ttf`) — TYPE LOUNGE 레이아웃 self-test 내장, 자간·여백은 `KERN`·`EMARGIN` 상수로 조정 후 재생성. (2026-07 `TYPE LOUNGE` 리네이밍: T·Y·P 글리프 교체, 레이어드 E는 TYPE의 마지막 글자로 동일 유지. T-Y 커닝·P-E 여백 튜닝.)
- 오렌지 배경에는 `-white`(화이트) 사용.
