MODE LOUNGE WORDMARK v10

겹침을 레터링 안에서 표현 — 글자가 서로 포개지고(투명 레이어), 잘리고(면 컷), 획을 공유합니다. NOMAD·HIDE·EATON 기법. 자간은 타이트하게(로고 밀도). 순수 잉크 모노크롬. "여러 용도가 하나의 공간에 겹쳐 든다"는 컨셉이 글자의 겹침 자체로 읽힙니다.

1 · 글자 포개짐 (Overlapping Layers)

글자가 반투명하게 서로 겹쳐 교집합이 짙어짐 — 멀티유즈가 한 자리에 포개진 모습
A1 — MODE 포개짐 (가로)
4글자가 35% 겹쳐 레이어로 쌓임 · 잉크 투명
A2 — MODELOUNGE 포개짐 (한 줄 리본)
10글자 전부 겹쳐 흐르는 띠
A1 — 다크 반전
잉크 위 크림, 스크린 블렌드

2 · 세로 포개짐 (Stacked Layers)

두 단어가 가로·세로로 동시에 겹쳐 밀도 높은 블록
B1 — MODE / LOUNGE 적층 겹침
줄 안에서 글자 겹침 + 줄끼리 세로 겹침
B2 — 공유 O (ML 두 단어가 O를 공유)
MODE·LOUNGE가 가운데 하나의 O를 함께 씀
B1 — 다크
잉크 배경

3 · 면 컷 · 네거티브 (Plane Cut)

솔리드 블록 글자를 사선 면이 가르며 겹침을 드러냄 — 건축적, 공간 브랜드와 결
C1 — MODE 블록 + 사선 컷
꽉 찬 글자를 사선 크림 면이 잘라 레이어 분리
C2 — MODE LOUNGE 블록 (2줄, 컷)
두 줄 솔리드, 사선 면이 둘을 가로질러 겹침
C3 — 가로 띠가 글자를 가로지름
수평 면이 워드마크 허리를 겹쳐 지남(멀티플라이)

4 · 환원 기하 · 모노그램 (Reduced / Monogram)

본질 형태로 환원하거나 ML이 획을 공유 — 다 안 그려도 읽힘
D1 — ML 모노그램 (획 공유)
M과 L이 한 획을 공유해 겹친 단일 마크
D2 — EATON식 환원 ('MODE'·O=점)
O를 솔리드 점으로 환원, 기하 스텐실
D3 — ML 모노그램 + 워드마크 락업
마크 + 작은 MODE LOUNGE (실사용 형태)
v10 — 겹침이 레터링 안에 내재 (v9의 '옆에 붙인 장치' 폐기)
  • 1 글자 포개짐(NOMAD) — 글자가 반투명 레이어로 겹침. 컨셉의 정수. (A1 MODE / A2 풀워드 / 다크)
  • 2 세로 포개짐 — 가로·세로 동시 겹침 블록 + 공유 O. (B1 적층 / B2 공유O)
  • 3 면 컷(HIDE) — 사선·수평 면이 글자를 갈라 겹침. 건축적. (C1·C2 사선 / C3 가로띠)
  • 4 환원·모노그램(EATON·DUO) — 본질 형태/획 공유. (D1·D3 ML 모노그램 / D2 O=점 환원)

이건 1차 재설계입니다. 끌리는 방향 1~2개 + 거슬리는 점을 알려주시면, 그 방향만 커스텀 레터폼으로 정밀 제도(획 두께·겹침 비율·커닝·곡률)해 진짜 로고 완성도로 끌어올리겠습니다. 확인: start logo-concepts-v10-overlap.html
