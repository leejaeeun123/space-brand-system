TYPE LOUNGE — 워드마크 컨셉 방향 3안

심볼을 접고 영문 워드마크 단독으로 다시 시작합니다. 공유해 주신 레퍼런스 7장이 공유하는 원리는 하나 — “의미를 별도 심볼이 아니라 글자 안에 담는다.” 그리고 TYPE LOUNGE에는 O가 두 개 있어 이 O가 전환·겹침·담기의 무대가 됩니다.

A. 두 개의 O Overlap · 겹침 / 연결 참조 · DUO / NORD CONNECT / again

mOde와 lOunge의 두 O를 주인공으로. 같은 원 유닛이 두 위치에서 겹치거나 이어진다 = “하나의 시스템(공간)이 여러 쓰임을 품는다.” 가장 직관적이고, 겹치는 교집합을 포인트 컬러/패턴 소스로 확장하기 좋습니다.

B. 타입 다이얼 O Switch · 전환 참조 · morning edition (+ 기존 ‘회전면’ 컨셉 계승)

O를 ‘상태가 바뀌는 다이얼/창(窓)’으로. 안쪽 면이 반원·분할로 채워지며 지금의 타입을 표시 = WORK · CLASS · GATHER. 폐기한 솔리드 팬의 ‘회전하는 면 = 전환되는 타입’ 아이디어를 심볼이 아니라 글자 안으로 가져오는 방향입니다.
