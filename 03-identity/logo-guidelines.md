## 1. 로고 구성

| 요소 | 내용 |
|---|---|
| 워드마크 | `MODE LOUNGE` (로마자, League Spartan Black 계열 기하 산세) + 국문 `모드라운지` 병기 |
| 심볼 | **겹쳐지는 면(Overlapping Planes)** — 여러 모드(WORK·CLASS·GATHER)를 나타내는 기하 면 2~3개가 겹침. 멀티플라이 교집합 = 라운지(공유의 핵) |
| 로케이션 락업 | `모드라운지 합정 / MODE LOUNGE HAPJEONG` |

## 2. 심볼 컨셉 — "겹침(Overlap)"

- **핵심**: 반투명 기하 면의 겹침. 각 면 = 하나의 모드, 겹친 부분 = 공유의 핵(라운지). 컨셉 명제 *"분위기는 하나, 용도는 여럿"* 이 한 도형에서 그대로 읽힘.
- **멀티플라이 교집합**: orange 면이 겹치며 교집합에서 색이 깊어진 **제3색**. (기하 컬러블록 문법과 일치)
- **형태**: 라운지의 수평감을 살린 둥근 사각/면 중심. 원도 보조로 쓸 수 있으나 **주인공 아님** — 겹침이 주인공.
- **위트**: 겹침의 실루엣에서 자연스럽게 'M'(MODE)가 떠오르게 (강제하지 않음).
- 보조 그래픽 디바이스: 컬러블록·유기적 블롭(furn. 방식), 지오메트릭 오너먼트(절제).
- **그루비 흐물 레터링은 사용 안 함** (정제된 가구와 톤 일치).

## 3. 로고 변형 (Variants)

| 변형 | 용도 |
|---|---|
| Primary (워드마크 가로) | 기본 |
| Stacked (2단 적재) | 정사각·SNS 프로필·간판 |
| Symbol only (겹침 면 모티프) | 파비콘·스탬프·픽토 |
| Location lock-up | 외부 간판·지도 |

## 4. 클리어 스페이스 & 최소 크기

- 클리어 스페이스: 심볼 면 높이(=X)의 **1X** 사방 여백.
- 최소 크기: 워드마크 높이 디지털 16px / 인쇄 8mm. 심볼 단독 12px / 6mm.

## 5. 컬러 운용 (레이어 분리)

| 맥락 | 로고 색 |
|---|---|
| 디지털·마케팅 | 컬러블록 위 — orange/white/ink 조합 과감하게 |
| 실제 공간 사이니지 | 블랙·화이트 물성 단색 (가구를 이기지 않게) |
| 사진 위 | 단색(화이트/잉크) + 충분한 대비 |

## 6. 금지 사항 (Misuse)

- ❌ 비율 왜곡·기울이기 / 그림자·외곽선·글로우
- ❌ 그루비·흐물 레터링으로 변형
- ❌ 브랜드 팔레트 외 색
- ❌ 겹침(교집합)을 없애고 면을 분리해 나열 — 겹침이 컨셉의 핵심
- ❌ 태양·광선(ray) 등 '선룸' 잔재 모티프 재도입
- ❌ 디자인 가구 제품 고유명을 로고/슬로건에 결합

## 7. 적용 우선순위

인스타 프로필(Stacked) → 외부 간판(Location lock-up) → 실내 사이니지(Symbol/블랙·화이트) → 안내문(Wordmark).
