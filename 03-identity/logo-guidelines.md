# Phase 3 — 로고 가이드라인 (v3)

> 2026-06-21 개정. 특정 이름 가정 표현 제거 → **네이밍 미정(name-agnostic)**. 'O'가 들어가는 이름(무드룸·오브룸·오빗 등)과 특히 잘 맞음. **'겹침(overlap)'을 핵심 모티프로 추가.** 무드보드 레퍼런스 추가.
> 미드센추리 그래픽(기하 산세) 기반. 방향·규칙 정의 — 벡터 제작은 승인 후.

## 1. 로고 구성

| 요소 | 내용 |
|---|---|
| 워드마크 | 확정 네이밍의 로마자(League Spartan Black 계열 기하 산세) + 국문 병기. 예시는 특정 이름이 아니라 **'O' 모티프 중심**으로 전개. 'O'가 들어가는 이름(무드룸·오브룸·오빗 등)과 특히 잘 맞음 |
| 심볼 | **'O'/원 + 겹침(overlap) 모티프** — 여러 쓰임이 한 공간에 포개진 멀티유즈를 'O'/원의 **겹침**으로 형태화. 'O'(또는 해)는 미드센추리 원(노구치·태양)으로. 번트 오렌지 = 태양과 직결 |
| 로케이션 락업 | `[브랜드명] 합정 / [BRAND] HAPJEONG` |

## 2. 심볼 컨셉

- **겹침(overlap)이 핵심 모티프** — 여러 쓰임(WORK·CLASS·GATHER)이 한 공간에 포개지는 멀티유즈를 'O'/원의 겹침으로 형태화. 원 두 개가 겹친 벤다이어그램·이클립스(반쯤 가린 원) 등으로 전개.
- 'O'/해를 **완전한 원**으로 — 바우하우스 기하 + 노구치 램프/태양의 MCM 곡선. 광선(ray) 디테일은 절제.
- 보조 그래픽 디바이스: 컬러블록·유기적 블롭(furn. 방식), 지오메트릭 오너먼트(절제).
- **그루비 흐물 레터링은 사용 안 함** (정제된 가구와 톤 일치).

## 3. 로고 변형 (Variants)

| 변형 | 용도 |
|---|---|
| Primary (워드마크 가로) | 기본 |
| Stacked (2단 적재) | 정사각·SNS 프로필·간판 |
| Symbol only ('O'/겹침 모티프) | 파비콘·스탬프·픽토 |
| Location lock-up | 외부 간판·지도 |

## 4. 클리어 스페이스 & 최소 크기

- 클리어 스페이스: 'O' 지름(=X)의 **1X** 사방 여백.
- 최소 크기: 워드마크 높이 디지털 16px / 인쇄 8mm. 심볼 단독 12px / 6mm.

## 5. 컬러 운용 (레이어 분리)

| 맥락 | 로고 색 |
|---|---|
| 디지털·마케팅 | 컬러블록 위 — orange/cream/ink/green/cobalt 조합 과감하게 |
| 실제 공간 사이니지 | 우드·브라스·블랙 물성 단색 (가구를 이기지 않게) |
| 사진 위 | 단색(크림/잉크) + 충분한 대비 |

## 6. 금지 사항 (Misuse)

- ❌ 비율 왜곡·기울이기 / 그림자·외곽선·글로우
- ❌ 그루비·흐물 레터링으로 변형
- ❌ 브랜드 팔레트 외 색
- ❌ 'O' 모티프를 임의 도형으로 대체
- ❌ 미드센추리 가구 제품 고유명을 로고/슬로건에 결합

## 7. 적용 우선순위

인스타 프로필(Stacked) → 외부 간판(Location lock-up) → 실내 사이니지(Symbol/우드·브라스) → 안내문(Wordmark).
