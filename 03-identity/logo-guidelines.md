# Phase 3 — 로고 가이드라인

> 1순위 가정 네이밍 **겸 (GYEOM)** 으로 예시 전개. 이름이 바뀌어도 구조·원칙은 유효합니다.
> 본 문서는 **방향·규칙**을 정의하며, 실제 벡터 제작은 승인 후 디자이너/툴로 진행합니다.

## 1. 로고 구성

| 요소 | 내용 |
|---|---|
| 워드마크 | `겸` (국문, 마루부리 계열 명조) + `GYEOM` (로마자, Fraunces 계열) |
| 심볼 | **겹침 마크** — 두 기하 도형(원 + 사각 또는 두 아크)이 겹쳐 '겸(兼)·멀티유즈'를 은유 |
| 로케이션 락업 | `겸 합정 / GYEOM HAPJEONG` (검색 변별력·확장성) |

## 2. 심볼 컨셉

- **겹침(overlap)** = 하나의 공간에 여러 쓰임이 포개진다는 핵심 메타포.
- 미드센추리 기하(바르셀로나의 직선 + 노구치의 원)를 차용한 **원+사각의 부분 겹침**.
- 보조 그래픽 디바이스: **리드드 글라스의 수직 선** 텍스처(반복 세로선) — 패턴·배경에 활용.

## 3. 로고 변형 (Variants)

| 변형 | 용도 |
|---|---|
| Primary (심볼 + 워드마크 가로 락업) | 기본 |
| Stacked (심볼 위 / 워드마크 아래) | 정사각·SNS 프로필 |
| Symbol only | 파비콘·스탬프·사이니지 픽토 |
| Wordmark only | 좁은 공간·푸터 |
| Location lock-up (겸 합정) | 외부 간판·지도 등록 |

## 4. 클리어 스페이스 & 최소 크기

- **클리어 스페이스**: 심볼 높이(=X)의 **1X** 만큼 사방 여백 확보. (절제·여백 원칙)
- **최소 크기**: 디지털 워드마크 높이 16px / 인쇄 8mm 이상. 심볼 단독 최소 12px / 6mm.

## 5. 컬러 운용

| 배경 | 로고 색 |
|---|---|
| 웜 크림(Oat) | 잉크 블랙 |
| 잉크 블랙 | 웜 크림(Oat) 또는 허니 오크 |
| 허니 오크 | 잉크 블랙 |
| 사진 위 | 단색(블랙/크림) + 가독 위해 충분한 대비 확보 |

- 1도 운용 기본. 크롬 실버는 **사이니지 등 물성 표현 시에만** 메탈릭으로.

## 6. 금지 사항 (Misuse)

- ❌ 비율 왜곡·기울이기
- ❌ 임의 색 적용(브랜드 팔레트 외)
- ❌ 그림자·외곽선·글로우 등 효과 추가
- ❌ 저대비 배경에 얹어 가독성 훼손
- ❌ 심볼·워드마크 간격 임의 변경
- ❌ 미드센추리 가구 **제품 고유명**을 로고/슬로건에 결합 (상표 가드레일)

## 7. 적용 우선순위

인스타 프로필(Stacked) → 외부 간판(Location lock-up) → 실내 사이니지(Symbol) → 안내문(Wordmark).
