## 1. 검증표 (상위 3후보)

| 후보 | 상표(KIPRIS) | 도메인 | 인스타 핸들 | 1차 소견 |
|---|---|---|---|---|
| **선룸 / SUN ROOM** | ⚠️ 확인 필요 | `sunroom.kr` / `sunroom.co.kr` 확인 | `@sunroom.hapjeong` 확인 | '선룸=유리온실' 일반어 → **도형 결합·로케이션으로 식별력 보강** 권장. 동명 카페/업체 가능성 점검 |
| 오빗 / ORBIT | ⚠️ 확인 필요 | `orbit.kr` 등 선점 가능성 높음 → 변형(`orbithapjeong`) 검토 | `@orbit.hapjeong` 확인 | 영문 일반어라 도메인 경쟁 치열. 변별은 좋으나 .com 확보 난이도↑ |

## 2. 사용자 직접 확인 체크리스트

- [ ] **상표**: [KIPRIS](http://www.kipris.or.kr) 후보별 + **제43류(공간대여)** · 제41류(클래스) 검색
- [ ] **도메인**: 후이즈/가비아에서 `.com/.co.kr/.kr` 가용성
- [ ] **인스타**: 핸들 점유 여부
- [ ] **네이버/구글**: 합정권 동일·유사 상호, 동명 공간 검색
- [ ] (노모룸 선택 시) 모회사 노모컴퍼니 상표권 범위 — 시리즈 출원 가능 여부

## 3. 식별력 보강 전략 (일반어 약점 대응)

선룸·오빗처럼 일반어성이 있으면:
1. **로케이션 결합**: 선룸 합정 / SUN ROOM HAPJEONG
2. **도형(로고) 결합 출원**: 워드마크 + 태양/원 심볼 동시 출원
3. **조어 변형**: 철자·표기 고유화

## 4. 전량 탈락 시 폴백

territory 교체(웜·선 → 모드 → 스페이스에이지 → 컨테이너 → 노모)로 재생성, 최대 3라운드 후 재협의.
