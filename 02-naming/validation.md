> 상표·도메인·SNS 핸들 가용성. 실시간 가용성은 라이브 확인 필요 → 1차 소견 + 사용자 체크리스트.

## 1. 검증표 (상위 3후보)

| 후보 | 상표(KIPRIS) | 도메인 | 인스타 핸들 | 1차 소견 |
|---|---|---|---|---|
| **노모룸 / NOMO ROOM** | ⚠️ 확인 필요 | `nomoroom.com` / `.co.kr` 확인 | `@nomo.room` 확인 | "노모"가 식별력 보유 → 단독보다 유리. 'room' 결합으로 변별 ↑ |
| 노모라운지 / NOMO LOUNGE | ⚠️ 확인 필요 | `nomolounge.com` 확인 | `@nomo.lounge` 확인 | 무난. lounge 일반어지만 노모로 식별 |
| 노모하우스 / NOMO HAUS | ⚠️ 확인 필요 | `nomohaus.com` 확인 | `@nomo.haus` 확인 | HAUS 철자로 변별·검색 유리 |

> ※ 모회사 **노모컴퍼니**의 기존 상표/법인 등록 상태를 먼저 확인하면, 공간 브랜드는
> 시리즈 상표(노모룸 등)로 묶어 출원하는 전략이 가능합니다.

## 2. 사용자 직접 확인 체크리스트

- [ ] **모회사 확인**: 노모컴퍼니의 상표 등록 분류·권리 범위 (시리즈 출원 가능 여부)
- [ ] **상표**: [KIPRIS](http://www.kipris.or.kr) "노모룸/NOMO ROOM" + **제43류(공간대여)** · 제41류(클래스)
- [ ] **도메인**: `nomoroom`, `nomolounge`, `nomohaus` + `.com/.co.kr/.kr`
- [ ] **인스타**: `@nomo.room` 등 핸들 점유 여부
- [ ] **네이버**: 합정권 동일/유사 상호 검색
