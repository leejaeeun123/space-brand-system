## 2. 사용자 직접 확인 체크리스트

- [ ] **상표**: [KIPRIS](http://www.kipris.or.kr) 후보별 + **제43류(공간대여)** · 제41류(클래스) 검색
- [ ] **도메인**: 후이즈/가비아에서 `.com/.co.kr/.kr` 가용성
- [ ] **인스타**: 핸들 점유 여부
- [ ] **네이버/구글**: 합정권 동일·유사 상호, 동명 공간 검색
- [ ] (노모룸 선택 시) 모회사 노모컴퍼니 상표권 범위 — 시리즈 출원 가능 여부

## 3. 식별력 보강 전략 (일반어 약점 대응)

무드룸·선룸·오빗·레이어처럼 일반어성이 있으면:
1. **로케이션 결합**: 무드룸 합정 / MOOD ROOM HAPJEONG
2. **도형(로고) 결합 출원**: 워드마크 + 'O'/원·겹침 심볼 동시 출원
3. **조어 변형**: 철자·표기 고유화

## 4. 전량 탈락 시 폴백

territory 교체(웜·선 → 모드 → 스페이스에이지 → 컨테이너 → 노모)로 재생성, 최대 3라운드 후 재협의.
