# Phase 2 — 네이밍 검증

> 상표·도메인·SNS 핸들 가용성 검증표. **실시간 가용성은 라이브 확인이 필요**하여,
> 본 표는 ① 컨셉/일반명사 충돌의 1차 평가와 ② 사용자가 직접 확인할 체크 항목을 제공합니다.

## 1. 검증표 (상위 2후보)

| 후보 | 상표(KIPRIS) | .com / .co.kr 도메인 | 인스타 핸들 | 1차 소견 |
|---|---|---|---|---|
| **겸 (GYEOM)** | ⚠️ 확인 필요 | `gyeom.co.kr` / `gyeomhapjeong.com` 확인 필요 | `@gyeom.hapjeong` 확인 필요 | 1음절 일반어라 단독 상표 식별력 약 → **로케이션·도형 결합 권장** |
| 여백 (YEOBAEK) | ⚠️ 확인 필요 | `yeobaek.space` / `.co.kr` 확인 필요 | `@yeobaek.space` 확인 필요 | 일반명사 → 식별력 보강(도형/접미) 필요. 동명 카페·브랜드 존재 가능성 높음 |

## 2. 사용자 직접 확인 체크리스트 (내일)

- [ ] **상표**: [KIPRIS](http://www.kipris.or.kr) 에서 "겸" / "여백" + 분류 **제43류(공간대여·임대)** · 제41류(교육·클래스) 검색
- [ ] **도메인**: 후이즈/가비아에서 `gyeom`, `yeobaek` + `.com/.co.kr/.kr/.space` 가용성
- [ ] **인스타그램**: `@gyeom.hapjeong`, `@yeobaek.space` 등 핸들 점유 여부
- [ ] **네이버**: 동일/유사 상호의 합정권 기존 업체 검색 (지역 충돌 방지)

## 3. 식별력 보강 전략 (일반명사 약점 대응)

단음절·일반명사는 상표 식별력이 약할 수 있어, 다음 중 하나로 보강 권장:

1. **로케이션 결합**: "겸 합정", "여백 합정" — 확장성(C5)과 동시 해결
2. **도형(로고) 결합 출원**: 워드마크 + 심볼을 함께 상표 출원해 식별력 확보
3. **조어 변형**: "겸(GYEOM)"을 그대로 두되 스타일라이즈드 레터링으로 고유성 부여

## 4. 전량 탈락 시 폴백 (계획 합의)

상위 후보가 상표/도메인 모두 충돌이면:
1. 기준 1축(예: "1~2음절" 제한) 완화 후 3·4·5순위(사이/결/마루/거실) 재검토 — **최대 3라운드**
2. 3라운드 후에도 실패 시 → 사용자와 네이밍 기준 재협의

## 5. 현재 상태

🟡 **컨셉 충돌 1차 평가 완료 / 법적·기술적 가용성 라이브 확인 대기**
→ 내일 체크리스트(2번) 확인 후 최종 확정 권장.
