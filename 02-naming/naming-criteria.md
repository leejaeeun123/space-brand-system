# Phase 2 — 네이밍 기준

## 1. 네이밍 컨셉

브랜드명은 **"분위기는 하나, 용도는 여럿"** 이라는 멀티유즈 정체성과,
**미드센추리·따뜻·절제** 의 무드를 동시에 담아야 합니다.

방향: 기능(스튜디오·대관)을 직접 말하기보다, **정서(거실·여백·겸용)** 를 은유하는 한국어 단어.

## 2. 평가 기준 (6축)

| # | 기준 | 설명 | 가중 |
|---|---|---|---|
| C1 | 컨셉 적합성 | 멀티유즈("겸용/유연/비움")를 은유하는가 | ★★★ |
| C2 | 무드 적합성 | 미드센추리·따뜻·절제와 어울리는가 | ★★★ |
| C3 | 기억·발음 | 짧고 부르기 쉽고 각인되는가 | ★★ |
| C4 | 검색 변별력 | 검색·해시태그에서 묻히지 않는가 | ★★★ |
| C5 | 확장성 | 로케이션/서브브랜드로 확장 가능한가 (예: "○○ 합정") | ★★ |
| C6 | 상표·도메인 | 등록 가능성·도메인·핸들 확보 가능성 | ★★★ (확인 필요) |

## 3. 네이밍 톤 가이드

- 한국어 단어 우선, 로마자 병기로 글로벌·SNS 대응 (예: 겸 / GYEOM).
- 1~2음절 선호 (각인). 단, 1음절은 검색 변별력(C4) 보강 장치 필요 → **로케이션 접미("○○ 합정")** 로 해소.
- 직접적 기능어(스튜디오·렌탈·스페이스) 회피 — 정서어 우선.
- 제품 고유명(미드센추리 가구 브랜드) 차용 금지.

## 4. 검증 절차 (validation.md)

후보 ≥5개 도출 → 6축 스코어링 → 상위안 **상표(KIPRIS)·도메인·SNS 핸들** 가용성 확인.
- 전량 탈락(상표/도메인 충돌) 시: 기준 1축(예: 음절 수) 완화 후 재생성, **최대 3라운드**.
- 3라운드 후에도 실패 시: 사용자와 기준 재협의.

> ⚠️ 상표·도메인·핸들의 **실시간 가용성은 수동/라이브 확인이 필요**합니다(KIPRIS, 후이즈, 인스타그램).
> 본 문서의 가용성 칸은 "확인 필요" 상태로 두고, 컨셉 충돌 여부만 1차 평가했습니다.
