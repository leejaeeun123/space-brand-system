# Phase 2 — 네이밍 기준 (v2)

> 2026-06-20 사용자 피드백 반영 전면 개정. 이전 후보(겸/여백 등) 전량 폐기.

## 1. 네이밍 컨셉

- **모회사 "노모컴퍼니(NOMO)"의 루트를 이어받는다.** 공간 브랜드 = `노모 + 공간 성격어` (예: 노모라운지).
- 무드: **미드센추리 모던 + 웜우드** 위에 **볼드 MCM 그래픽**(바우하우스·기하) 감성.

## 2. 평가 기준 (6축)

| # | 기준 | 설명 | 가중 |
|---|---|---|---|
| C1 | 노모 루트 일관 | "노모(NOMO)"를 명확히 품는가 | ★★★ |
| C2 | 2음절 이상 | 1음절 단독 금지 (사용자 요구) | 필수 |
| C3 | 멀티유즈 적합 | 사무실·클래스·모임을 다 품는 그릇인가 | ★★★ |
| C4 | MCM 무드 | 미드센추리·기하 그래픽 톤과 어울리는가 | ★★ |
| C5 | 기억·발음 | 짧고 부르기 쉽고 로고화 좋은가 | ★★ |
| C6 | 상표·도메인 | 등록·도메인·핸들 가용성 | ★★★ (확인 필요) |

## 3. 네이밍 구조

```
[노모 / NOMO]  +  [공간 성격어: 라운지 · 룸 · 하우스 · 살롱 · 클럽 · 플로어 …]
```

- 로마자 락업이 로고의 주인공(레퍼런스의 "THE SPIN ROOM"처럼 영문 워드마크 중심).
- 국문 병기: `노모라운지` / 로마자 `NOMO LOUNGE`.
- 성격어는 **멀티유즈를 좁히지 않는 중립 그릇**일수록 유리 (룸 > 라운지 > 클럽).

## 4. 톤 가이드

- 그루비/지오메트릭 볼드 레터링이 어울리는 단어 (각진·둥근 글자 조형이 살아나는).
- 기능어(스튜디오·렌탈) 직접 노출은 지양하되, "룸/라운지"는 레트로 컨테이너로 허용.
- 미드센추리 가구 제품 고유명 차용 금지.
