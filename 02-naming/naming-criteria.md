# Phase 2 — 네이밍 기준 (v3)

> 2026-06-20 재개정. "노모 루트 필수" 해제 → **독립 네이밍도 허용**. 노모 연계는 보너스(필수 아님).

## 1. 네이밍 컨셉

- 멀티유즈 라운지 **"분위기는 하나, 용도는 여럿"** 을 담는 이름.
- 무드: **미드센추리 모던 + 웜우드**, 표현은 **MCM 그래픽**(바우하우스·기하).
- 모회사 노모컴퍼니 연계는 **선택**(시리즈 전략 시 가점). 굳이 '노모'를 넣지 않아도 됨.

## 2. 평가 기준 (6축)

| # | 기준 | 설명 | 가중 |
|---|---|---|---|
| C1 | 멀티유즈 적합 | 사무실·클래스·모임을 다 품는 그릇인가 | ★★★ |
| C2 | 2음절 이상 | 1음절 단독 금지 | 필수 |
| C3 | 무드 적합 | 미드센추리·웜우드·기하 그래픽과 어울리는가 | ★★★ |
| C4 | 로고화 | 'O'/원/태양 등 기하 모티프로 풀기 좋은가 | ★★ |
| C5 | 기억·발음·검색 변별 | 짧고 부르기 쉽고 묻히지 않는가 | ★★ |
| C6 | 상표·도메인 | 등록·도메인·핸들 가용성 | ★★★ (확인 필요) |
| (보너스) | 노모 연계 | 모회사 시리즈로 묶이는가 | + |

## 3. 네이밍 territory (탐색 축)

| 축 | 설명 | 예 |
|---|---|---|
| **컨테이너** | 용도를 안 정하는 '방/공간' | 룸·살롱·로비·플로어·하우스 |
| **모드** | 멀티모드(WORK/CLASS/GATHER) 직결 | 모드룸 |
| **웜·선(sun)** | 따뜻함·빛·미드센추리 글로우 | 선룸·데이룸 |
| **스페이스에이지** | 60s 미드센추리 우주시대·원형 | 오빗·써클 |
| **노모 연계** | 모회사 루트(선택) | 노모룸·노모라운지 |

## 4. 톤 가이드

- 기하 산세 볼드 레터링이 살아나는 단어(또렷한 글자 조형).
- 기능어 직접 노출 지양하되 '룸/라운지/로비'는 레트로 컨테이너로 허용.
- 미드센추리 가구 제품 고유명 차용 금지.
