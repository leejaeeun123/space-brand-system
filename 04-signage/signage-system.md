# Phase 4 — 사이니지 시스템 (개요, v2)

> 층고·정확한 입면 치수 `[대기]` → **상대 규격**으로 정의, 절대치는 `[[placeholder]]`.
> 3층 입지라 외부 노출 제한 → 사이니지가 "발견하게 만드는" 핵심 역할.
> 1순위 가정 네이밍: **선룸 / SUN ROOM**.

## 1. 사이니지 원칙

1. **레이어 분리** — 외부·창문(마케팅 성격)은 컬러블록 과감하게, **실내 사이니지는 우드·브라스·블랙 물성 중심**으로 차분히(가구를 이기지 않게).
2. **MCM 그래픽** — 기하 산세 + 컬러블록 + 'O' 모티프. 그루비 배제.
3. **모드 가변성** — WORK/CLASS/GATHER를 안내에 반영.
4. **찾아오게 만들기** — 1층부터 3층까지 끊김 없는 동선 유도.

## 2. 사이니지 체계

| 분류 | 항목 | 문서 |
|---|---|---|
| 외부 | 블레이드(돌출) 사인, 창문 사인, 1층 디렉토리 | `exterior-signage.md` |
| 실내 | 입구 사인, 모드 안내, 웨이파인딩, 에티켓 | `interior-wayfinding.md` |
| 시스템 | 픽토그램(모드·편의·방향) | `pictograms.md` |

## 3. 물성·컬러 (design-tokens 참조)

| 요소 | 물성 | 컬러 토큰 |
|---|---|---|
| 실내 메인 사인 | 월넛/오크 우드 또는 블랙 도장 스틸 | `material/walnut` · `spine/ink` |
| 레터/디테일 | 브러시드 브라스(놋쇠) | `material/brass` |
| 외부·창문(마케팅) | 컬러 도장/아크릴 | `spine/orange` · `block/green` · `block/cobalt` |
| 크림 면 | 크림 도장/아크릴 | `spine/cream` |

## 4. 타이포 (사이니지)

- 디스플레이: League Spartan(기하 산세) / 국문 Pretendard Black
- 안내·픽토 라벨: Inter / Pretendard
- 모드 태그: `WORK · CLASS · GATHER`

## 5. 멀티유즈 모드 시나리오 (≥3, 페르소나 대응)

| 모드 | 페르소나 | 사이니지가 전달할 것 |
|---|---|---|
| **WORK** | 1인 워커 | 작업 모드, 와이파이·콘센트, 정숙 |
| **CLASS** | 클래스 호스트 | 클래스 세팅, 가구 재배치, 수용 인원 |
| **GATHER** | 모임 주최자 | 모임 모드, 이용 에티켓 |

→ 상세 → `interior-wayfinding.md`.
