# Phase 4 — 사이니지 시스템 (개요)

> 층고·정확한 입면 치수 `[대기]` → **상대 규격**으로 정의, 절대치는 `[[placeholder]]`.
> 3층 입지라 외부 노출이 제한적 → 사이니지가 "발견하게 만드는" 핵심 역할.

## 1. 사이니지 원칙

1. **절제된 안내** — 정보는 명료하게, 장식은 최소(브랜드 보이스 '절제').
2. **물성으로 무드 전달** — 우드 + 블랙 스틸 + 크롬 + 한지광. 가구 팔레트와 동일 물성.
3. **모드 가변성** — WORK/CLASS/GATHER 모드를 안내에 반영(멀티유즈 정체성).
4. **찾아오게 만들기** — 3층 동선을 1층부터 끊김 없이 유도.

## 2. 사이니지 체계 (종류)

| 분류 | 항목 | 문서 |
|---|---|---|
| 외부 | 블레이드(돌출) 사인, 창문 사인, 1층 입구 디렉토리 | `exterior-signage.md` |
| 실내 | 입구 사인, 모드 안내, 웨이파인딩(화장실·동선), 에티켓 | `interior-wayfinding.md` |
| 시스템 | 픽토그램(모드·편의·방향) | `pictograms.md` |

## 3. 공통 물성·컬러 (design-tokens 참조)

| 요소 | 물성 | 컬러 토큰 |
|---|---|---|
| 메인 사인 베이스 | 오크/자작 원목 또는 블랙 도장 스틸 | `wood/honey-oak` / `ink/black` |
| 레터링 | 무광 블랙 또는 크림 | `ink/black` / `base/oat` |
| 메탈 디테일 | 헤어라인 스테인리스 | `metal/chrome` |
| 백릿(필요 시) | 한지/아크릴 확산판 | `glow/paper` |

## 4. 타이포 (사이니지)

- 디스플레이: Maru Buri / Fraunces (브랜드·간판)
- 안내·픽토 라벨: Pretendard / Inter (고가독)
- 모드 태그: `WORK · CLASS · GATHER` (Fraunces, 대문자, 자간 +0.08em)

## 5. 멀티유즈 모드 시나리오 (≥3, 페르소나 대응)

| 모드 | 페르소나 | 사이니지가 전달할 것 |
|---|---|---|
| **WORK** | 1인 워커 | 작업 모드, 와이파이·콘센트, 정숙 안내 |
| **CLASS** | 클래스 호스트 | 클래스 세팅 가이드, 가구 재배치 동선, 수용 인원 |
| **GATHER** | 모임 주최자 | 모임 모드, 이용 에티켓, 분위기 해치지 않는 절제 안내 |

→ 상세 시나리오는 `interior-wayfinding.md`.

## 6. 규격 표기 규칙

- 절대 치수가 필요한 항목은 `[[placeholder: 층고/입면 확정 후 입력]]` 로 표기.
- 상대 규격(비율·위계)만 우선 확정. 예: "블레이드 사인은 창문 사인의 1/3 폭".
