# Phase 4 — 사이니지 시스템 (개요, v2)

> 2026-06-21 개정. 층고 **약 2m(낮은 편) 확정** 반영 — 낮은 층고에 맞춘 사인 높이·돌출 규격. 운영 = **무인 자동화** → 스태프 없는 **자가안내(self-guide)가 핵심**. 네이밍 미정(name-agnostic). 무드보드 레퍼런스 추가.
> 3층 입지라 외부 노출 제한 → 사이니지가 "발견하게 만드는" 핵심 역할.

## 1. 사이니지 원칙

1. **레이어 분리** — 외부·창문(마케팅 성격)은 컬러블록 과감하게, **실내 사이니지는 우드·브라스·블랙 재질(물성) 중심**으로 차분히(가구를 이기지 않게).
2. **MCM 그래픽** — 기하 산세 + 컬러블록 + 겹침(overlap) 모티프. 그루비 배제.
3. **모드 가변성** — WORK/CLASS/GATHER를 안내에 반영.
4. **찾아오게 만들기** — 1층부터 3층까지 끊김 없는 동선 유도.
5. **자가안내(self-guide)** — 무인 자동화 운영이라 현장 스태프가 없음. **직관적 도형·컬러블록 길찾기**로 입실·이용·퇴실을 사용자가 스스로 끝낼 수 있게. 예약·결제는 앱, 현장 안내는 사이니지가 전담.
6. **낮은 층고(약 2m) 대응** — 사인 높이는 눈높이 이하로 압축, 천장 돌출은 최소화. (→ §6 규격)

## 5. 멀티유즈 모드 시나리오 (≥3, 페르소나 대응)

| 모드 | 페르소나 | 사이니지가 전달할 것 |
|---|---|---|
| **WORK** | 1인 워커 | 작업 모드, 와이파이·콘센트, 정숙 |
| **CLASS** | 클래스 호스트 | 클래스 세팅, 가구 재배치, 수용 인원 |
| **GATHER** | 모임 주최자 | 모임 모드, 이용 에티켓 |

→ 상세 → `interior-wayfinding.md`.

## 6. 규격 표기 규칙 (층고 약 2m 반영)

- **층고 약 2m 확정** → 낮은 층고 기준 규격:
  - 시선 높이 사인: 바닥에서 **약 1.4~1.6m**(눈높이) 중심, 상단이 천장에 닿지 않게.
  - 천장 돌출·행잉 사인: **최소화** 또는 벽부착으로 대체(머리 충돌·답답함 방지). 불가피하면 바닥에서 1.9m 이상 확보 곤란 → 행잉 지양.
  - 외부 블레이드(돌출) 사인: 실내 층고와 무관(건물 외벽) → 건물 규정 내에서 별도. `[[placeholder: 건물 외부 사인 규정]]` 유지.
- 입면·창 규격 등 측정 필요한 절대치는 `[[placeholder]]` 유지.
- 상대 규격(비율·위계)은 우선 확정.
