# Phase 4 — 픽토그램 시스템

> 미드센추리 기하 라인 스타일. 일관된 그리드·선 굵기로 "고른(curated)" 인상.
> 실제 벡터 제작은 승인 후 — 여기서는 **스타일 규칙 + 목록 + 의미**를 정의.

## 1. 픽토그램 스타일 규칙

| 속성 | 값 |
|---|---|
| 그리드 | 24×24 기준, 2px 패딩. **납품 SVG는 viewBox 100×100** |
| 선 굵기 | 3 균일(viewBox 100 기준, 중앙정렬), 끝단 **square** · 조인 **miter** |
| 형태 | 기하 + 약한 곡선 (미드센추리), 직각은 `radius/sm`(2px) |
| 컬러 | `spine/ink` 기본 / 반전 시 `spine/cream` / 컬러블록 위 가능. 파일은 `currentColor` |
| 채움 | 라인 위주 인상 유지 — 단, **납품 SVG는 stroke를 아웃라인(면)으로 변환** (아래 7절) |
| 톤 | 절제 — 디테일·장식 금지(V2/V7) |

## 2. 모드 픽토그램 (3종, 시그니처)

| 픽토 | 모드 | 모티프 |
|---|---|---|
| 🖥 (라인) | **WORK** | 데스크 + 램프 라인 (집중) |
| 🧘 (라인) | **CLASS** | 매트 + 사람 곡선 (요가/클래스) |
| ☕︎ (라인) | **GATHERING** | 둘러앉은 점·아크 (모임) |

> 위 이모지는 의미 표시용 placeholder. 실제는 통일된 24px 라인 픽토로 제작.

## 3. 편의·방향 픽토그램

| 분류 | 항목 |
|---|---|
| 편의 | 와이파이, 콘센트, 화장실, 탕비/물, 전신거울, 옷걸이 |
| 방향 | 화살표(상/하/좌/우), 입구/출구 — **사각형을 쌓아 만든 도형 조립 화살표**(미드센추리 톤) |
| 안내 | 정숙, 금연, 신발, 퇴실 정리 |
| 자가안내(무인) | 예약·앱 안내, QR, 입실/퇴실 절차, 결제 — 스태프 없는 셀프 이용 가이드 |

> 무인 자동화 운영이라 **방향·자가안내 픽토가 특히 중요** — 도형 화살표·컬러블록이 글자보다 먼저 길을 안내(→ signage-system.md 무드보드).

## 4. 픽토그램 + 라벨 병행 (필수)

- 모든 픽토그램은 **텍스트 라벨과 함께** 사용 (접근성 — 색·기호만으로 정보 전달 금지).
- 라벨: Paperlogy(국·영문 통일), `label/md`(14px).

예:
```
[wifi]  와이파이
[plug]  콘센트
[wc]    화장실 →
```

## 5. 적용 매핑

| 픽토 | 사용처 |
|---|---|
| 모드 3종 | I-2 모드 안내판, 창문 모드 태그, 인스타 |
| 편의 | I-3 이용 가이드, I-4 웨이파인딩 |
| 방향 | 외부 유도 사인, 실내 동선 (도형 조립 화살표) |
| 안내 | I-5 에티켓 카드 |
| 자가안내(무인) | 입구 디렉토리·I-3 이용 가이드 — 앱/QR·입퇴실 절차 |

## 6. 제작 산출물 (승인 후)

- 24px 라인 픽토 세트(SVG), 모드 3종 + 편의/방향/안내.
- 단색·반전 2버전. 모드 안내판용 교체 카드(WORK/CLASS/GATHERING) 디자인.

## 7. 납품 벡터 — 아웃라인 (2026-07-12)

> 인쇄·소형 사이니지(세로 5~10mm)를 위해 **모든 stroke를 채운 아웃라인(면) 패스로 변환**.
> stroke 렌더링·폰트 의존이 없어 어떤 크기·환경에서도 선 굵기가 정확히 비례한다.

### 편의·방향·안전 픽토 (9종, `picto-*.svg`)
`coffee · extinguisher · no · nosmoke · room · stairs · toilet · umbrella · wifi`
- viewBox 100×100, `fill="currentColor"` 단일 패스, `fill-rule="evenodd"`(닫힌 링은 자동으로 구멍).
- 원본 stroke 사양(굵기 3·square 캡·miter 조인·중앙정렬)을 형태 손실 없이 재현.
- 색은 CSS `color`/`fill`로 제어 — 잉크·크림·컬러블록 위 모두 사용.
- 재현 스크립트: `outline_picto.py` (shapely 버퍼).

### 모드 태그 (9종, `mode-{work,class,gathering}[-ink|-white].svg`)
네모 박스 + 모드명 라벨. **큐레이션 교체 카드·창문 태그·안내판**용.
- 서체 **Paperlogy 400(Regular)** 아웃라인 — 소형(5~10mm)에서 600·500은 두꺼워 본문과 같은 400로. (400은 브랜드 2-웨이트 내)
- 대문자 · 자간 **+0.05em** · 직각 모서리 · **테두리 4pt 아웃라인**(stroke 아님, 프레임 패스).
  - 라벨 스펙(+0.12em)보다 좁힘 — 박스 안 소형 라벨에선 넓은 자간이 성겨 보여 조정. 자간은 `gen_modes.py`의 `TRACK_EM` 환경변수로 재조정 가능.
- 박스 높이 3종 동일 → 나란히 배치 시 baseline·라인 정렬.
- 색 3종: `currentColor`(웹) / `-ink`(#16130F, 라이트 배경) / `-white`(#ffffff, 잉크·오렌지 배경).
- 재현 스크립트: `gen_modes.py` (fontTools 글리프 → 패스).
