# Phase 4 — 픽토그램 시스템

> 미드센추리 기하 라인 스타일. 일관된 그리드·선 굵기로 "고른(curated)" 인상.
> 실제 벡터 제작은 승인 후 — 여기서는 **스타일 규칙 + 목록 + 의미**를 정의.

## 1. 픽토그램 스타일 규칙

| 속성 | 값 |
|---|---|
| 그리드 | 24×24 기준, 2px 패딩 |
| 선 굵기 | 1.5px 균일 (스트로크), 끝단 라운드 최소 |
| 형태 | 기하 + 약한 곡선 (미드센추리), 직각은 `radius/sm`(2px) |
| 컬러 | `spine/ink` 기본 / 반전 시 `spine/cream` / 컬러블록 위 가능 |
| 채움 | 라인 위주, 면 채움 최소 |
| 톤 | 절제 — 디테일·장식 금지(V2/V7) |

## 2. 모드 픽토그램 (3종, 시그니처)

| 픽토 | 모드 | 모티프 |
|---|---|---|
| 🖥 (라인) | **WORK** | 데스크 + 램프 라인 (집중) |
| 🧘 (라인) | **CLASS** | 매트 + 사람 곡선 (요가/클래스) |
| ☕︎ (라인) | **GATHER** | 둘러앉은 점·아크 (모임) |

> 위 이모지는 의미 표시용 placeholder. 실제는 통일된 24px 라인 픽토로 제작.

## 3. 편의·방향 픽토그램

| 분류 | 항목 |
|---|---|
| 편의 | 와이파이, 콘센트, 화장실, 탕비/물, 전신거울, 옷걸이 |
| 방향 | 화살표(상/하/좌/우), 입구/출구 — **사각형을 쌓아 만든 도형 조립 화살표**(미드센추리 톤) |
| 안내 | 정숙, 금연, 신발, 퇴실 정리 |
| 자가안내(무인) | 예약·앱 안내, QR, 입실/퇴실 절차, 결제 — 스태프 없는 셀프 이용 가이드 |

> 무인 자동화 운영이라 **방향·자가안내 픽토가 특히 중요** — 도형 화살표·컬러블록이 글자보다 먼저 길을 안내(→ signage-system.md 무드보드).

## 4. 픽토그램 + 라벨 병행 (필수)

- 모든 픽토그램은 **텍스트 라벨과 함께** 사용 (접근성 — 색·기호만으로 정보 전달 금지).
- 라벨: Pretendard(국·영문 통일), `label/md`(14px).

예:
```
[wifi]  와이파이
[plug]  콘센트
[wc]    화장실 →
```

## 5. 적용 매핑

| 픽토 | 사용처 |
|---|---|
| 모드 3종 | I-2 모드 안내판, 창문 모드 태그, 인스타 |
| 편의 | I-3 이용 가이드, I-4 웨이파인딩 |
| 방향 | 외부 유도 사인, 실내 동선 (도형 조립 화살표) |
| 안내 | I-5 에티켓 카드 |
| 자가안내(무인) | 입구 디렉토리·I-3 이용 가이드 — 앱/QR·입퇴실 절차 |

## 6. 제작 산출물 (승인 후)

- 24px 라인 픽토 세트(SVG), 모드 3종 + 편의/방향/안내.
- 단색·반전 2버전. 모드 안내판용 교체 카드(WORK/CLASS/GATHER) 디자인.
