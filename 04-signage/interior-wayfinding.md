# Phase 4 — 실내 안내판 & 웨이파인딩

> 52.49㎡ 단일층(라운지 존 + 룸 존). 작은 공간이므로 사인은 **소수·정밀·절제**.
> 모드 가변성을 반영한 안내가 핵심.
> 2026-06-21 개정. 층고 약 2m(낮은 편) 확정 → 사인 높이 구체화. 무인 자동화 운영 → **자가안내가 사인의 1차 임무**(스태프 응대 대체).

## 1. 실내 사인 인벤토리

| # | 사인 | 위치 | 물성 | 내용 |
|---|---|---|---|---|
| I-1 | 입구 사인 | 현관 | 월넛 우드 + 브라스 레터링 | 'O'/겹침 심볼 + `[브랜드명] 합정` + 환영 한 줄 |
| I-2 | 모드 안내판 | 라운지 진입부 | 블랙 스틸 + 교체형 카드 | `오늘의 모드: WORK / CLASS / GATHER` |
| I-3 | 이용 가이드 | 잘 보이는 벽면 | 크림 아크릴 + 무광 블랙 | 와이파이·콘센트·이용시간·연락 |

| I-5 | 에티켓 | 출입구·퇴실 동선 | 크림 카드 | 정숙·정리·퇴실 안내(절제 톤) |

## 2. 모드별 안내 시나리오 (멀티유즈 핵심)

### WORK 모드 (1인 워커)
- 모드판: `WORK — 집중하는 라운지`
- 안내: 와이파이 `[[placeholder: SSID/PW]]`, 콘센트 위치, "통화는 복도에서" 정숙 안내
- 가구: 다이닝테이블/데스크 + 프루베풍 체어 세팅

### CLASS 모드 (클래스 호스트)
- 모드판: `CLASS — 비우면 스튜디오`
- 안내: 가구 재배치 가이드(가벽·스툴을 벽으로), 전신거울 활용, 수용 인원 `[[placeholder]]`
- 안전: 매트 구역, 동선 확보 안내

### GATHER 모드 (모임 주최자)
- 모드판: `GATHER — 둘러앉는 라운지`
- 안내: 라운지 소파·데이베드·확장 테이블 세팅, 무드 조명(페이퍼램프) 사용, 이용 에티켓
- 절제: "분위기를 위해 안내는 최소로" — 큰 경고문구 대신 작은 카드

## 3. 웨이파인딩 원칙

- **Recognition over recall**: 픽토그램 + 짧은 라벨 병행(색만으로 정보 전달 금지).
- 작은 공간이므로 사인 **개수 최소화** — 꼭 필요한 5종만.
- 사인 높이·위치(층고 약 2m 확정): 시선 높이 사인은 바닥에서 **약 1.4~1.6m** 중심. **천장 행잉 사인은 지양**(낮은 층고 → 머리 충돌·답답함). 벽부착·바닥 사인으로 대체.
- **자가안내 우선**: 무인 운영이라 사인이 곧 안내 데스크. 입실→이용→퇴실 전 단계를 사인만으로 완결할 수 있게, **직관적 도형·컬러블록 길찾기**로 구성(글자보다 도형이 먼저).

## 4. 접근성 (KWCAG 준용)

- [ ] 픽토그램에 텍스트 라벨 병행 (색 무관 정보 전달)
- [ ] 명도 대비 4.5:1 이상 (spine/ink on spine/cream 충족)
- [ ] 화장실·비상 동선 명확 표기
- [ ] 한글 + 필요한 영문 병기

## 5. 교체형(가변) 사인 메커니즘

- 모드 안내판(I-2)은 **카드 슬롯/마그넷 교체형** — 운영자가 모드 전환 시 카드만 교체. (무인 운영 시 사용자/원격 전환 또는 앱 연동 표시 고려.)
- 카드 3종(WORK/CLASS/GATHER) 사전 제작. 픽토그램 + 모드명 + 한 줄.
