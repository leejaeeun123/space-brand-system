# Phase 4 — 외부 사이니지

> 핵심 제약: **합정역 앞 3층** — 1층 노출 없음, 보행자 시선은 지상. "찾아오게 만드는" 유도가 관건.
> 치수는 상대 규격 우선, 절대치는 `[[placeholder]]` (층고·입면·건물 규정 확정 후).

## 1. 외부 사인 3종

### ① 블레이드(돌출) 사인 — 발견의 핵심
- **역할**: 보행자가 길에서 3층의 존재를 인지하게 함.
- **위치**: 건물 외벽, 보행 동선 직각 방향 돌출(양면).
- **디자인**: 블랙 도장 스틸 프레임 + 크림/오렌지 레터링 + 'O' 심볼. 야간 백릿 옵션.
- **내용**: 심볼 + `노모룸 / NOMO ROOM` + `3F`.
- **규격**: `[[placeholder: 건물 외부 사인 규정·돌출 허용 치수 확인]]` / 비율 = 세로형, 심볼:워드 = 1:1.

### ② 창문(파사드) 사인 — 무드 전달
- **역할**: 3층 창에서 브랜드 무드를 노출, 호기심 유발.
- **디자인**: 컬러블록(오렌지/크림) + 기하 워드마크 시트 + 'O' 모티프. 창문은 마케팅 레이어라 컬러 과감 허용.
- **내용**: 워드마크 + `WORK · CLASS · GATHER` 모드 태그.
- **규격**: 창 폭의 1/2 이내, 시선 높이 우선.

### ③ 1층 입구 디렉토리 / 유도 사인
- **역할**: 1층에서 3층까지 **끊김 없는 동선** 확보(엘리베이터·계단 안내).
- **디자인**: 우드 또는 블랙 스틸 명패 + 층 표기. 건물 디렉토리 등재(`노모룸 합정 3F`).
- **내용**: `노모룸 합정 · 3F` + 방향 화살표 픽토(→ pictograms).
- **규격**: 건물 공용부 규정 준수 `[[placeholder]]`.

## 2. 동선 유도 시퀀스

```
[길 위 보행자] → 블레이드 사인 인지 → [건물 앞] 입구 디렉토리 → 엘리베이터/계단 유도 → [3F 도착] 입구 사인
```
- 각 접점에 최소 1개 사인. **3층까지 한 번도 끊기지 않게.**

## 3. 야간 시인성

- 블레이드·창문 사인은 **반전 운용**(블랙 배경 + 크림/페이퍼 글로우)으로 야간 가독 확보.
- 저녁 GATHER 모드 시간대 가동 고려 — 야간 노출이 중요.
