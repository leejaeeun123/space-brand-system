# Phase 1.5 — 무드보드 검증 게이트

> **목적**: 확보된 가구 무드(가구-의존 층)와 비주얼 원칙(가구-무관 층)이 모순 없이 정합하는지 검증.
> **통과 규칙**: 아래 매핑표의 **충돌 셀 0개 = 통과**. 통과 전까지 Phase 3 컬러/소재 확정값 금지(placeholder만).
> **충돌 정의**: 가구 무드 키워드가 비주얼 원칙의 **회피 방향(X1–X7)과 직접 모순**되는 경우.
> **판정자**: 별도 verifier (작성자 자기판정 금지) — 본 문서 하단 판정 기록 참조.

## 1. 매핑표 (가구 무드 M1–M7 ↔ 비주얼 형용사 V1–V7)

| 가구 무드 (discovery) | 매핑되는 비주얼 형용사 | 충돌하는 회피선? | 충돌 |
|---|---|---|---|
| M1 미드센추리 모던 | V3 클래식, V1 따뜻 | X3(트렌디) 회피와 정합 | ✅ 없음 |
| M2 웜 라이트우드 | V1 따뜻, V7 고른 | X1(차가운 미니멀) 회피와 정합 | ✅ 없음 |
| M3 블랙 가죽 | V4 정밀, V2 절제(무게중심 앵커) | 회피선과 모순 없음 | ✅ 없음 |

| M5 웜 페이퍼·린넨 | V5 부드러운 빛, V1 따뜻 | X5(쨍한 직사광) 회피와 정합 | ✅ 없음 |

**충돌 셀 카운트: 0**

## 2. 해소된 긴장 (충돌 아님, 기록용)

- **웜 vs 쿨 (M2 우드 ↔ M3·M4 블랙/크롬)**: 미드센추리 양식은 본래 따뜻한 우드 + 블랙 가죽 + 크롬을 함께 씀. 충돌이 아니라 **웜 베이스 + 쿨 액센트**의 의도된 대비. → Phase 3 팔레트에서 비율로 통제(우드/크림 다수, 블랙/크롬 소수).

## 3. 확정된 팔레트 방향 (게이트 산출물 → Phase 3 입력)

게이트 통과 시 Phase 3이 받는 **방향성**(확정 색값 아님):
- 베이스: 웜 라이트우드 + 웜 크림 (다수 면적)
- 앵커: 잉크 블랙 (텍스트·무게중심)
- 액센트: 코냑(가죽) · 크롬 실버 (소수)
- 빛: 웜 페이퍼 글로우
