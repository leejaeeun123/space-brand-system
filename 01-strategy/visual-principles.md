# Phase 1 — 비주얼 원칙 (가구-무관 층)

> 컨셉·보이스만으로 도출 가능한 비주얼 방향입니다. 실제 색값/소재(가구-의존 층)는
> Phase 1.5 무드보드 게이트 통과 후 Phase 3에서 확정합니다. 여기서는 **방향과 회피선**만 정의합니다.

## 1. 비주얼 형용사 (지향)

브랜드 비주얼이 풍겨야 할 7가지 형용사 — Phase 1.5 게이트에서 가구 무드 키워드와 매핑됩니다.

| # | 형용사 | 의미 |
|---|---|---|
| V1 | **따뜻한 (warm)** | 차갑지 않은, 우드 톤 기반의 온기 |
| V2 | **절제된 (restrained)** | 여백이 많은, 요소를 덜어낸 |
| V3 | **클래식한 (timeless)** | 유행 타지 않는, 미드센추리의 항구성 |
| V4 | **정밀한 (precise)** | 메탈·기하의 또렷한 디테일 |
| V5 | **부드러운 빛 (soft-lit)** | 페이퍼 램프 같은 확산광의 분위기 |
| V6 | **유연한 (modular)** | 재구성 가능함을 시각적으로 암시 |
| V7 | **고른 (curated)** | 아무거나가 아닌, 의도된 선택 |

## 2. 회피 방향 (안티-패턴)

| # | 회피 | 이유 |
|---|---|---|
| X1 | 차가운 미니멀(병원·테크 화이트) | V1 따뜻함과 충돌 |
| X2 | 과한 장식·플로럴·러스틱 | V2 절제와 충돌 |
| X3 | 트렌디·Y2K·네온 | V3 클래식과 충돌 |
| X4 | 흐릿·뭉개진 핸드드로잉 일색 | V4 정밀함과 충돌 |
| X5 | 쨍한 직사광·고채도 원색 | V5 부드러운 빛과 충돌 |
| X6 | 고정적·권위적 그리드(관공서) | V6 유연함과 충돌 |
| X7 | 잡다한 클립아트·이모지 남발 | V7 큐레이션과 충돌 |

## 3. 형태 언어 (방향)

- **기하 + 유기의 균형**: 미드센추리 특유의 직선(프루베·바르셀로나)과 곡선(노구치·임스)의 공존.
- **여백 우선 레이아웃**: 요소를 꽉 채우지 않고, 한 화면에 하나의 주인공.
- **그래픽 디바이스 후보**: 리드드 글라스의 수직 선 텍스처, 겹침(overlap)으로 '겸/멀티유즈' 은유.

## 4. 톤 & 무드 (방향)

- 전반적으로 **로우 콘트라스트 + 웜 뉴트럴** 베이스에 **블랙으로 또렷한 앵커**.
- 사진은 **자연광·확산광**, 그림자가 부드러운 미드센추리 인테리어 톤.

## 5. 타이포 방향 (확정은 typography.md)

- **세리프(따뜻·클래식) + 산세리프(정밀·중립)** 의 페어링.
- 국문은 따뜻한 명조 계열을 디스플레이로, 모던 고딕을 본문으로.
