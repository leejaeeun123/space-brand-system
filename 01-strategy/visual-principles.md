# Phase 1 — 비주얼 원칙 (v3, MCM 그래픽)

> 2026-06-20 확정. 코어 무드를 **기하 그래픽**(바우하우스·스위스·기하)으로 표현.
> 볼드·따뜻한 컬러는 유지하되 **그루비/사이키델릭은 배제** — 정제된 가구와 같은 DNA.

## 1. 비주얼 형용사 (지향)

| # | 형용사 | 의미 |
|---|---|---|
| V1 | **볼드·웜 (bold & warm)** | 자신감 있는 따뜻한 원색, 큰 조형 |
| V2 | **기하 (geometric)** | 원·사각·블롭의 또렷한 도형, 바우하우스 |
| V3 | **정제된 (refined)** | 볼드하되 난잡하지 않은, 스위스적 질서 |
| V4 | **물성 있는 (tactile)** | 블랙 + 화이트의 실제 질감 |
| V5 | **위트 (restrained wit)** | 면이 겹쳐 생기는 교집합 같은 작은 재치 (그루비로 흐르지 않음) |

## 2. 회피 방향 (안티-패턴)

| # | 회피 | 이유 |
|---|---|---|
| X1 | 차가운 테크 미니멀(SaaS 화이트) | V1 따뜻·볼드와 충돌 |
| X2 | 그루비·사이키델릭·Cooper류 흐물 레터링 | V3 정제와 충돌 (가구와 톤차) |
| X3 | 뮤트·소심한 파스텔·그레이지 일색 | V1 볼드와 충돌 |
| X4 | 디테일 과잉·러스틱 장식 | V2 기하와 충돌 |
| X5 | 납작한 플랫·물성 없는 일러스트 일색 | V4 물성과 충돌 |

## 3. 형태 언어

- **컬러블록 + 기하/블롭 셰이프**: 화면을 색면으로 과감히 분할(furn./RETROTIKA). 블롭은 솔 바스·노구치식 유기적 MCM 곡선(그루비 아님).
- **빅 지오메트릭 타이포**: 기하 산세 디스플레이가 그래픽의 주인공.
- **레트로 지오메트릭 오너먼트**: 겹치는 면·기하 도형·반복 패턴을 **절제해서** 포인트로.
- **역할 분리**: 쨍한 컬러는 배경/그래픽, 블랙·화이트는 피사체/물성.

## 4. 레이어 분리 (중요)

| 레이어 | 톤 |
|---|---|
| 디지털·마케팅 (인스타·포스터·웹) | 컬러블록 과감하게 — 볼드 MCM 그래픽 |
| 실제 공간 사이니지 | 블랙·화이트 물성 중심 — 차분하게(가구를 이기지 않게) |

## 5. 톤 & 무드

- 하이 콘트라스트 + 웜 새추레이션. 번트 오렌지 베이스 + 블랙 앵커.
- 사진은 디자인 가구를 색면·블롭 위에 얹는 **컷아웃 콜라주**(furn. 방식).

## 6. 타이포 방향 (확정은 typography.md)

- 디스플레이: **기하 산세(바우하우스/푸투라 계열)** 볼드. 세리프 이탤릭은 강조 1곳만.
- 본문: 스위스 그로테스크(고가독).
