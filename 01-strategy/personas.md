# Phase 1 — 페르소나

> 운영 모델 `[확정]` = **무인 자동화 운영(스태프 없는 셀프 이용)**. 특정 타입에 상주 인력이 붙지 않으므로
> 3종 페르소나를 **동등 가중**으로 유지합니다(상주 스태프가 특정 타입을 밀어줄 수 없음 → 한쪽으로 쏠 근거 없음).
> 무인 운영 특성상 **자가안내(self-guide) 니즈가 전 페르소나 공통으로 강해집니다** — 현장 응대 대신 사이니지·앱이 안내를 전담.
> (Phase 4 사이니지 타입 시나리오와 1:1 대응)

## 페르소나 1 — 1인 워커 / 소규모 팀 (사무실 타입)

| 항목 | 내용 |
|---|---|
| 프로필 | 프리랜서·1인 사업자·원격 근무자, 2030. 합정·홍대 인근 거주/활동 |
| Job-to-be-done | "집중되면서 분위기 좋은, 카페보다 사적인 작업 공간이 필요하다" |
| 페인포인트 | 카페는 시끄럽고 눈치 보임 / 공유오피스는 삭막하고 월 단위 부담 |
| 기대 | 좋은 책상(아레초·프루베 체어), 조용함, 시간 단위 결제, 콘센트·와이파이 |
| 사이니지 니즈 | 작업 타입 안내, 와이파이/콘센트/이용시간, 정숙 안내 |

## 페르소나 2 — 클래스 호스트 (요가·원데이 클래스 타입)

| 항목 | 내용 |
|---|---|
| 프로필 | 요가·필라테스 강사, 공예·드로잉·향수 등 원데이 클래스 운영자, 2040 |
| Job-to-be-done | "가구를 옆으로 빼면 매트를 깔 수 있는, 사진 잘 나오는 클래스 공간이 필요하다" |
| 페인포인트 | 전용 스튜디오는 임대료 부담 / 일반 대관은 분위기가 안 나옴 |
| 기대 | 가변 레이아웃(가벽·스툴 이동), 따뜻한 조명, 전신거울, 바닥 컨디션 |
| 사이니지 니즈 | 클래스 타입 세팅 가이드, 가구 재배치 안내, 인원 수용 안내 |

## 페르소나 3 — 모임 주최자 (소개팅·소셜 모임 타입)

| 항목 | 내용 |
|---|---|
| 프로필 | 소개팅·소셜다이닝·북클럽·생일 모임 주최자, 2030 |
| Job-to-be-done | "어색하지 않게 대화가 트이는, 우리끼리 편한 거실 같은 공간이 필요하다" |
| 페인포인트 | 카페·식당은 옆 테이블 신경 쓰임 / 파티룸은 너무 '파티'스러움 |
| 기대 | 라운지 소파·데이베드, 확장 테이블, 무드 조명, 프라이버시(가벽) |
| 사이니지 니즈 | 모임 타입 안내, 공간 이용 에티켓, 분위기를 해치지 않는 절제된 안내 |

## 공통 인사이트

- 셋 모두 **"분위기는 이미 완성, 나는 콘텐츠만"** 을 원함 → 메시징 필러 1("가져오기만 하세요")과 직결.
- 셋 모두 **사진/SNS** 를 의식 → 공간과 사이니지가 "찍고 싶게" 디자인되어야 함.
- 셋의 시간대가 다름(낮 워크 / 저녁 클래스 / 주말 모임) → **가동률 순환** 설계 가능.
- **무인 자동화 운영** → 셋 모두 현장에 물어볼 스태프가 없음. 입실·이용·퇴실을 **스스로 끝낼 수 있어야** 함 → 자가안내(앱 예약·결제 + 사이니지 길찾기/타입 가이드)가 페르소나 공통의 1순위 니즈.
