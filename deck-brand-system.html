로고·디스플레이·본문 모두 Paperlogy — 각진 기하 골격으로 “정돈된” 키워드를 타이포로 표현. 영문 자간 −2%, 본문 400·600.

Application 01

명함

두꺼운 코튼지에 평압 잉크. 앞면 세로형 워드마크, 뒷면 연락처·주소. 종이 물성으로 “편안함”을 전한다.

Application 02

1층 입간판

철제 프레임 + 반투명 아크릴 패널. 계단형 화살표로 3층 유도, WORK·CLASS·GATHERING 모드 안내.

Application 03

대문 명판

반투명 frosted 아크릴에 잉크 각인. 벽에서 살짝 떠 그림자를 만드는 스탠드오프 설치.

Application 04

화장실 사이니지

frosted 반투명 판. 픽토그램은 주황 포인트, 글씨·구분선은 잉크. 각진 모노라인 픽토.

Application 05

안내문 · 레이어드 보드

자석 보드에 트레이싱지·종이 안내문을 겹쳐 붙이고 주황 띠지로 포인트 — “넘나드는·레이어드” 컨셉의 물성 구현.
