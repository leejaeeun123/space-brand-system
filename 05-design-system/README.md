## 핵심 원칙 (4)

1. 유연성을 핵심 자산으로 — 용도는 여럿, 분위기는 하나
2. 인테리어에서 비주얼을 도출 — 미드센추리 무드가 진실의 원천
3. 문서가 진실의 원천 — 소비처 있는 항목만 토큰화
4. 계획/실행 분리 — 그래픽 실제 제작은 별도 승인

## 시그니처 요소

- **모드 태그**: `WORK · CLASS · GATHER` — 멀티유즈 정체성을 한 줄로.
- **겹침 심볼**: 두 도형이 포개져 '겸/멀티유즈'를 은유.
- **리드드 글라스 디바이스**: 세로선 패턴 — 배경·사이니지 텍스처.
