## 핵심 원칙 (5)

1. 유연성을 핵심 자산으로 — 용도는 여럿, 분위기는 하나
2. 인테리어에서 비주얼을 도출 — 미드센추리·웜우드가 진실의 원천
3. **MCM 그래픽 앵커** — 볼드하되 정제됨(가구와 같은 바우하우스 DNA), 그루비 배제
4. **레이어 분리** — 디지털·마케팅은 컬러풀, 공간 사이니지는 우드·브라스·블랙 물성
5. 계획/실행 분리 — 그래픽 실제 제작은 별도 승인

## 시그니처 요소

- **모드 태그**: `WORK · CLASS · GATHER` — 멀티유즈를 한 줄로.
- **'O' + 겹침(overlap) 모티프**: 'O'를 미드센추리 원(노구치·태양)으로, 여러 쓰임이 한 공간에 포개진 멀티유즈를 원의 겹침으로. ('O'가 들어가는 이름과 특히 잘 맞음.)
- **컬러블록 + 블롭**: 색면 분할 + 유기적 셰이프(furn. 방식).
