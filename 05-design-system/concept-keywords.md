# 타입라운지 — 컨셉 3키워드 & 표현 시스템

> **🟢 2026-07-05 확정 (재은 정리).** 이 문서가 디자인 시스템의 **코어 프레임워크**다.
> 이전 로고 탐색(겹침/감각 재프레이밍, `01-strategy/experience-sensations.md`)의 흐름은 이 3키워드로 **수렴**됨.
> 원본 정리 이미지: `assets/` (mode lounge 컨셉맵).

## 0. 대전제

**타입라운지 = 멀티유즈 라운지.** 한 공간에서 **일(WORK) · 학습(CLASS) · 교류(GATHER)** 가 모두 가능.
이 “멀티유즈”를 3개의 키워드로 나눠 표현한다.

```
        타입라운지 (멀티유즈 라운지)
   [WORK 일] — [CLASS 학습] — [GATHER 교류]
                    ↓
   정돈된  ·  넘나드는  ·  편안함
```

## 1. 정돈된 (큐레이션 · 안목 · 갖춰짐)

무엇을 하든 **정리되고 갖춰진** 공간. 안내 위주라 명료해야 한다.

| 축 | 방향 |
|---|---|
| 형태 | **각진 형태**, 사각형 그래픽 |
| 레이아웃 | **심플 · 가운데 정렬**(위/아래 배치) |
| 폰트 | **산세리프(Paperlogy)** — 각진 기하 골격 |
| 로고 | 심플 산세리프 워드마크 `TYPE LOUNGE` (심볼 없음) |

→ 타이포·로고·레이아웃 규칙은 `03-identity/typography.md`, `logo-system.md`.

## 2. 넘나드는 (자유로운 · 멀티 · 여러 용도)

한 공간이 여러 타입으로 **전환·중첩**되는 감각. **투명·레이어드(쌓기)** 로 표현.

| 축 | 방향 |
|---|---|
| 시각 | **반투명 · 투명**, 겹쳐 쌓는 **레이어드** |
| 로고 내장 | TYPE의 **E = 가로 3선(쌓인 층)** → 워드마크에 “레이어드” 내장. 3선 = WORK·CLASS·GATHER |
| 사이니지 | 공간 안내판에 **자석으로 여러 레이어를 쌓는** 방식 |
| 소재 | **반투명 아크릴** 사이니지 |
| 각인 대체 | 비용 문제로 각인 대신 **투명 스티커** 부착 또는 **트레이싱지 인쇄** 부착 |

## 3. 편안함 (아늑 · 따뜻 · 오래 머물 수 있는)

일·학습이 주 이용이라 **문구(objet)** 를 가장 많이 쓴다 → **‘종이’ 물성**으로 편안함을 표현.

| 축 | 방향 |
|---|---|
| 소재 | **종이** — 질감이 도드라지는 두꺼운 종이 또는 **반투명 트레이싱지** 인쇄 |
| 이미지 톤 | 또렷하지 않고 **한 겹 쌓인(레이어드) 느낌** |
| 포인트 컬러 | 오렌지 · 그린 · 블루 → **스티커**로 표현 |

## 4. 가장 필요한 것 — 문구(objet) 활용

편안함·레이어드를 동시에 살리는 **핵심 표현 도구**. 공간·인쇄물의 꾸밈 요소로 상시 운용.

- **클립 · 더블클립 · 포스트잇 · 실 · 자 · 자석 · 마스킹 테이프**
- 용도: 종이/아크릴 레이어를 **쌓고 고정**하는 실제 기능 + 브랜드 무드를 내는 **장식** 이중 역할.
- 포인트 컬러 마스킹 테이프·스티커로 **포인트**를 준다.

## 5. 키워드 ↔ 표현 매핑 요약

| 키워드 | 형태/시각 | 소재 | 대표 장치 |
|---|---|---|---|
| 정돈된 | 각진·사각·중앙정렬 | — | Paperlogy 워드마크, 사각 그래픽 |
| 넘나드는 | 반투명·레이어드 | 아크릴 | 로고 3선, 자석 레이어 사이니지 |
| 편안함 | 종이 물성·한 겹 느낌 | 종이·트레이싱지 | 문구 오브제, 포인트 스티커 |
